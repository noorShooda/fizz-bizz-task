## Fizz Bizz Task

This project is a solution for the FizzBuzz task.
The FizzBuzz task is : write a program that prints the numbers from 1 to 100. But for multiples of three print "Fizz" instead of the number and for the multiples of five print "Buzz". For numbers which are multiples of both three and five print "FizzBuzz".

## Steps to run the project

### `npm install`

### `npm start`

Runs the app in the development mode.<br />

The page will reload if you make edits.<br />
